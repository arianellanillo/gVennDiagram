gVennDiagram

============

A simple Venn Diagram's Union, Intersect and Not implementation in C# WPF


I wrote it for fun :D , enjoy!

*Environment :* .net 4.0 C# WPF @ VS2010

*What Supported :* Union, Intersect, Not
